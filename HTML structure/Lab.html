ให้หาว่า attribute ของ tag ต่อไปนี้มีอะไรบ้างและใช้งานอย่างไร

ชื่อ Tag Attribute การใช้งาน
<h> title,id title : ระบุข้อมูลเพิ่มเติมเกี่ยวกับองค์ประกอบ
id : เป็นการนำค่า id ที่เราสร้างไว้มาเรียกใช้งาน
<p> style style : สามารถเรียกใช้ CSS แบบอินไลน์ได้
<div> class class : สามารถเรียกใช้ class ที่เราสร้างไว้มาเรียกใช้งาน
<img> src,alt src : ระบุตำแหน่งของรูป
alt : ระบุข้อความหรือคำอธิบายของรูปภาพ
<a> href,rel href : ระบุ URL ของหน้าที่ลิงก์ไป
rel : ระบุความสัมพันธ์ระหว่างเอกสารปัจจุบันและเอกสารที่เชื่อมโยง
<ul> style style : สามารถเรียกใช้ CSS แบบอินไลน์ได้
<ol> start start : สามารถะบุค่าเริ่มต้นของรายการ
<table> style style : สามารถเรียกใช้ CSS แบบอินไลน์ได้
<form> action,method action : ระบุตำแหน่งที่จะส่งข้อมูลแบบฟอร์มเมื่อส่งแบบฟอร์ม
method : ระบุวิธี HTTP ที่จะใช้เมื่อส่งแบบฟอร์มข้อมูล
<button> type type : ระบุประเภทของปุ่ม

ให้หาความเหมือนและความแตกต่างระหว่าง 2 tag ที่กำหนดให้

ชื่อ Tag ความเหมือน ความต่าง
<hr> และ <br> เป็นการขึ้นบรรทัดใหม่ hr จะเป็นการขีดเส้น ส่วน br จะขึ้นบรรทัดใหม่เลย
<ul> และ <ol> จะเป็นการสร้างลิสต์รายการ ul จะมีจุดข้างหน้ารายการ ส่วน ol จะมีเลขลำดับข้างหน้ารายการ
<select> <checkbox>และ <radio button> จะเป็นการเลือกลิสต์รายการ select จะเป็นแบบ drop-down checkbox เป็นการเลือกแบบการติ๊กถูก radio button เป็นการเลือกแบบการติ๊กที่วงกลม
<div> และ <span> เป็นการจัดกลุ่มข้อมูล div จะเป็นแบบ block เลเวล ส่วน span จะเป็นแบบ inline เลเวล
<select> <checkbox>และ <radio button> จะเป็นการเลือกลิสต์รายการ select จะเป็นแบบ drop-down checkbox เป็นการเลือกแบบการติ๊กถูก radio button เป็นการเลือกแบบการติ๊กที่วงกลม
<script> และ <style> เป็นการเรียกใช้ funtion ที่เรากำหนด script เรียกใช้ javascript ส่วน style เรียกใช้ css

ให้หาว่า input type มีอะไรบ้างแต่ละ type แตกต่างกันอย่างไร
